MC MOBILE MINECRAFT HOST V2

SETUP:

1. Install Termux
2. Run:
   bash termux/install.sh

3. Start server:
   bash termux/start.sh

4. Start tunnel:
   bash termux/playit.sh

5. Start panel:
   cd panel_backend
   npm install
   node index.js

6. Open Android app (WebView)

IMPORTANT:
- Phone must stay ON
- Use PLAYIT for public IP
- Bedrock edition recommended
- Performance depends on RAM/CPU
